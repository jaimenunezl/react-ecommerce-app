# Store App

The application shows a list of products, which can be added to the cart and then proceed to checkout

## Requirements

- Node.js 18.x

## Tools

- [vite](https://vitejs.dev/)

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

## Deploy

```bash
npm run deploy
```
